# X19 辅助工具集 · 成品仓库

> 本仓库**仅用于存储**已编译打包好的可执行程序（成品文件），**不包含源代码**。  
> 项目面向 **网易 X19 电脑端** 玩家、服主、地图作者，提供完全免费、开箱即用的辅助工具。

---

## 📦 仓库内容

本仓库目前包含以下两款独立工具（均以 Release 压缩包形式提供）：

| 工具名称 | 功能简述 |
|----------|----------|
| **Login-Pro** | 联机大厅房间管理器（创建/监控/玩家准入/日志） |
| **Map-Manager** | 地图资源管理器（导入/导出/分类/版本回滚/冲突检测） |

两款工具互不依赖，可按需单独下载使用。

## 🛠 Login-Pro（联机大厅房间管理）

专为网易 X19 联机大厅设计，帮助玩家和服主高效管理多人游戏房间。

### 主要功能
- **房间生命周期管理**：一键创建、删除、修改房间配置（密码、人数、可见性等）。
- **实时状态监控**：可视化展示房间状态（等待/游戏中/关闭），自动检测异常掉线并提供快速恢复。
- **玩家准入控制**：支持白名单/黑名单过滤，基于 X19 ID 或角色名封禁/解封。
- **日志与审计**：完整记录房间操作日志，便于追溯和排障。
- **多实例支持**：同时管理多个 X19 账号下的联机大厅。

### 适用场景
- 个人快速开房、邀请好友。
- 社群统一管理多房间，避免冲突。
- 大型服务器自动化运维，降低人工成本。

---

## 🗺 Map-Manager（地图及存档综合管理）

专为网易 X19 地图作者、模组制作者和服务器管理员设计，提供一系列深度管理功能，简化存档与模组维护流程。

### 主要功能
- **账号登录管理**  
  支持多 X19 账号的登录信息存储与切换，方便在不同账号下进行操作，无需重复输入凭证。

- **模组监听下载转换**  
  自动监听指定来源的模组更新，支持模组文件的自动下载、格式转换（如版本适配），确保所用模组始终保持最新且兼容。

- **存档解密**  
  针对加密的游戏存档文件提供解密功能，便于玩家或管理员查看和编辑存档内容（在合法合规范围内使用）。

- **存档简便装载模组**  
  提供一键式模组装载到存档的功能，无需手动修改配置文件，大幅降低模组安装门槛，提升体验。

- **注册文件管理**  
  生成、备份和恢复必要的注册表项或配置文件，保证工具与游戏的正常协作，解决因注册信息缺失导致的运行异常。

### 适用场景
- 地图作者快速解密和编辑测试存档。
- 模组制作者自动化管理模组版本和转换。
- 服务器管理员批量装载模组到多个存档。
- 普通玩家轻松管理账号和资源，无需手动处理复杂文件。

## 💻 系统要求

- **操作系统**：Windows 10/11（64 位）
- **运行时环境**：**必须安装 [.NET Runtime 10.0.7 (x64)](https://dotnet.microsoft.com/en-us/download/dotnet/10.0)**，否则程序将无法启动。  
  请从微软官方下载并安装 `dotnet-runtime-10.0.7-win-x64.exe`。
- **硬盘空间**：每个工具压缩包约 100~200 MB（解压后，不含游戏文件或缓存数据）。
- **网络**：本地操作无需联网验证（Login-Pro 需正常访问 X19 联机服务，Map-Manager 完全离线）。

## 📥 下载与使用

### 1. 安装运行时（首次必须）
在运行任何工具之前，请先下载并安装 **.NET Runtime 10.0.7 (x64)**：  
👉 [https://dotnet.microsoft.com/en-us/download/dotnet/10.0](https://dotnet.microsoft.com/en-us/download/dotnet/10.0)  
选择 **Run desktop apps** 下的 `dotnet-runtime-10.0.7-win-x64.exe` 安装即可。

### 2. 获取成品
从本仓库的 **[Releases](https://github.com/tingting-bro/Storage_warehouse/releases)** 页面下载最新稳定版压缩包（`.zip` 或 `.7z`）。  
> ⚠️ Releases 中会同时提供 `Login-Pro` 和 `Map-Manager` 的独立压缩包，请按需下载。

### 3. 快速启动
- 解压对应工具的压缩包到任意目录（建议路径不含中文或空格）。
- 双击运行 `Login-Pro.exe` 或 `Map-Manager.exe`。
- 首次启动：
  - **Login-Pro** 会提示输入 X19 Cookie 授权（仅本地存储，不上传）。
  - **Map-Manager** 会自动检测 X19 游戏安装路径（若未找到，请手动指定）。
- 进入主界面后即可开始使用。
