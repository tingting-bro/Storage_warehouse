# Login-Pro

## 项目简介

**Login-Pro** 是一款专为 **网易 X19** 游戏平台设计的联机大厅房间管理工具。它旨在帮助玩家、服主和社区管理员更高效、更稳定地管理多人游戏房间，提升联机体验。

本项目完全 **免费** ，**严禁任何形式的盈利行为**（包括但不限于售卖、捆绑付费、植入广告收费等）。我们坚持社区共享精神，所有功能均面向广大玩家无偿提供。

---

## 主要功能

- **房间生命周期管理**  
  一键创建、删除、修改联机大厅房间配置，支持自定义房间名称、密码、最大人数、游戏模式等参数。

- **实时状态监控**  
  可视化展示当前所有房间的运行状态（等待中、游戏中、已关闭），并自动检测异常掉线房间，提供快速恢复机制。

- **玩家准入控制**  
  支持白名单/黑名单过滤，可基于 X19 用户 ID 或角色名进行快捷封禁/解封，有效防止恶意捣乱。

- **日志与审计**  
  记录所有房间操作日志（创建者、时间、变更内容），便于纠纷追溯和运维排障。

- **多实例支持**  
  可同时管理多个 X19 账号下的联机大厅，适合大型社区或服务器集群。

## 适用场景

- 个人玩家：快速开房，邀请好友，无需反复手动设置。
- 小型社群：统一管理多个房间，避免冲突和重复创建。
- 大型服务器：自动化运维，降低人工干预成本，提高稳定性。

## 安装与使用

### 下载
请从本仓库 [Releases](https://github.com/tingting-bro/Login-Pro-save-box/releases) 页面获取最新稳定版压缩包。

### 快速启动
1. 解压到任意目录。
2. 根据您的操作系统运行 `Login-Pro.exe`（Windows）
3. 首次启动会提示输入 X19 Cookie 账号授权（仅本地存储，不会上传）。
4. 进入主界面后，点击“新建房间”即可开始管理。
